# French

Primary active-study language on Tue/Thu per [[../PLAN]].

Given you already have working French, treat this folder as targeted improvement rather than ground-up acquisition: capture vocab you keep forgetting, grammar you never quite nailed, phrases you hear on InnerFrench that surprise you.

## Subfolders

- `journal/` — your daily writing in French (create when you first write an entry)

## Seed notes

- [[éclair]] — linked to [[../concepts/Lightning]]
- [[faux-amis]] — list of "false friends" with Spanish, grown over time

## Making new notes

Use [[../templates/vocabulary-fr|the French vocab template]]. If a word has a useful Spanish cognate, add the bridge section — but never put Spanish in the SR prompt itself.

## Review

Lunch slot on Tue/Thu. Only cards tagged `#sr/fr`. Don't review `#sr/es` in the same session.
