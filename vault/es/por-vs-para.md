# por vs para

por vs para::por = reason/cause/through; para = purpose/destination/recipient #sr/es

## Quick split

- **para** — goal, destination, recipient, deadline. *Forward-pointing.*
  - *Este regalo es para ti.* — This gift is for you.
  - *Salgo para Madrid.* — I'm leaving for Madrid.
- **por** — cause, reason, exchange, duration, through/along. *Backward- or around-pointing.*
  - *Gracias por tu ayuda.* — Thanks for your help.
  - *Pagué 20€ por el libro.* — I paid 20€ for the book.

## Memory aid

*para* → purpose, *por* → because. Both start with P; the vowel tells you which.

## French bridge

French uses *pour* for both, mostly, with *par* handling some of the "through/via" cases. Rough map:

- ES *para* (goal/destination) → FR *pour* — *C'est pour toi* = *Es para ti*
- ES *por* (cause/reason) → FR *pour* — *Merci pour ton aide* = *Gracias por tu ayuda*
- ES *por* (through/via) → FR *par* — *par la fenêtre* = *por la ventana*
- ES *por* (exchange) → FR *pour* — *20€ pour le livre* = *20€ por el libro*

**So French *pour* splits into two Spanish words.** If you're French-speaking, the rule "when in doubt, if French says *pour*, pick between *por* and *para* by checking if it's a reason (*por*) or goal (*para*)" gets you ~80% right.
