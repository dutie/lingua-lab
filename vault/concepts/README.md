# Cross-language concepts

**Navigation only. Files in this folder NEVER get `#sr` tags.**

When you're studying a word in one language and wonder how the others handle the same idea, come here. Don't make flashcards from these files — reviewing three translations of one English concept trains translation-mediated retrieval and creates interference between the languages you're studying simultaneously.

This folder is for your conscious curiosity, not your review queue.

## Contents

- [[Lightning]] — weather vocabulary (visible flash vs. bolt vs. thunder) across FR/ES/JA
- [[ser-estar-être]] — how Spanish's essence/state split compares to French's unified *être*

Grow this folder slowly. One bridge note only when you've actually bumped into the cross-language question twice — not preemptively.
