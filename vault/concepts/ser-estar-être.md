# ser / estar / être

**Cross-language reference map. Not a flashcard. Never add an `#sr` tag to this file.**

## The divergence

French uses one verb — *être* — for "to be" in all senses. Spanish splits this into two verbs, *ser* and *estar*, along an essence/state axis. This is the single biggest grammar gap between the two Romance languages for your purposes.

## The mapping

| Sense | French | Spanish |
|---|---|---|
| Identity/nationality | *Je suis belge* | *Soy belga* (ser) |
| Profession | *Je suis ingénieur* | *Soy ingeniero* (ser) |
| Time of day | *Il est trois heures* | *Son las tres* (ser) |
| Location | *Je suis à Madrid* | *Estoy en Madrid* (estar) |
| State/mood | *Je suis fatigué* | *Estoy cansado* (estar) |
| Ongoing action | *Je suis en train de lire* | *Estoy leyendo* (estar) |

## How to use this bridge

When you're constructing a Spanish sentence and your French instinct is to pick *être*, **pause and ask which Spanish verb applies**. The fastest heuristic: if the thing is a state (how someone/something IS right now) or a location, use *estar*. Otherwise use *ser*.

This heuristic won't give you the tricky cases (*es aburrido* "he is boring" vs *está aburrido* "he is bored"). Those you learn by exposure. See [[../es/ser-vs-estar]].
