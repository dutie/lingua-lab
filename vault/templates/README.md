# Templates

Copy one of these when starting a new note. Rename the file to match the target-language word.

Obsidian has a built-in "Templates" core plugin: Settings → Core plugins → Templates → set folder to `templates/`. Then `Cmd/Ctrl-P → Insert template` pastes into a blank note. If that feels like overhead, just open the template file and copy its contents manually.

## When to use which

| Template | Use when |
|---|---|
| `vocabulary-es` | Adding a Spanish word |
| `vocabulary-fr` | Adding a French word |
| `vocabulary-ja` | Adding a Japanese word |
| `grammar-point` | A grammar rule (any language) — examples: por-vs-para, subjunctive, ser-vs-estar |
| `concept-bridge` | Navigating between the same idea across languages — no SR tag! |
| `journal` | Daily 3-5 sentence writing in target language |

## The templates never get SR'd themselves

All template files are named with a template prefix or live only in `templates/`. The `#sr` lines inside them are examples, not active flashcards — the plugin will pick them up if you're not careful. If you notice templates polluting your review queue, add `templates/` to the Spaced Repetition plugin's "Folders to ignore" setting.
